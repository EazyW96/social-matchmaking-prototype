# Service Profile & Description

## Service Overview
* **Service Name:** Social Matchmaking G.E.S
* **Service Provider:** Group 2
* **Service Consumer:** Gamers looking for high-quality, non-toxic group finding (LFG).

## ## Service Description
This web service acts as a **Trusted Provider** by aggregating skill data from external APIs (Riot/Steam) and 

generating proprietary "Reputation Tags". It follows the **REST architectural style** to ensure stateless, scalable 

communication between the gaming client and the backend. Following the principles of Service-Oriented Architecture,

this service provides a clear interface for consumers to access verified player data.
