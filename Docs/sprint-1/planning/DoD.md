# Definition of Done (DoD)

A User Story or Task is only considered **DONE** when:
* **Code Quality:** The code is written in **JavaScript (Node.js)** and pushed to the `main` branch.
* **Test-First Evidence:** A failing "Red Phase" test case was recorded before any implementation logic was written.
* **Testing Coverage:** The feature has at least **10 passing unit tests** and **1 passing BDD/A-TDD test**.
* **Peer Review:** Evidence of **Pairing or Mobbing** (e.g., a photo or video) is documented in the GitHub repo.
* **Documentation:** The README and API specifications are updated to reflect the new increment.
